# Website Customization Intake Form

A beautiful client intake form for collecting website customization details.

## Quick Start

1. **Deploy to Netlify**
   - Drag and drop the `client-intake-form` folder to [Netlify](https://app.netlify.com/)
   - Get your deployed URL

2. **Update Success Redirect**
   - Open `index.html`
   - Find line with `redirect` value
   - Replace `https://yourdomain.netlify.app/success.html` with your actual URL

3. **Done!**
   - Form submissions will be sent to the configured email
   - Works immediately, no confirmation needed

## Form Submission

This form uses **Web3Forms** - a reliable form backend service.

- ✅ Works immediately (no email confirmation needed)
- ✅ All fields are optional
- ✅ Submissions sent instantly
- ✅ No rate limits for basic usage
- ✅ Works on localhost and live servers
